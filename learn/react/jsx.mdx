<LearnSectionContent>
JSX is a <strong>JavaScript syntax extension</strong> that make developer capable of writing HTML code inside React components.

It's very powerful feature and simplifies writing web applications

Example of using JSX:

</LearnSectionContent>

```javascript
function SomeComponent(props) {
  return <h1>Hello {props.name}</h1>
}
```
