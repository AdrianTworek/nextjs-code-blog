<LearnSectionContent>
  Example of HOF (higher-order function):
</LearnSectionContent>

```javascript
const numbers = [1, 2, 3, 4, 5, 6]

const modifiedNumbers = numbers.map((num) => num * 2)
console.log(modifiedNumbers) // [2, 4, 6, 8, 10, 12]

const filteredNumbers = modifiedNumbers.filter((num) => num % 4 === 0)
console.log(filteredNumbers) // [4, 8, 12]

const sumOfFilteredNumbers = filteredNumbers.reduce((acc, curr) => {
  return acc + curr
}, 0)
console.log(sumOfFilteredNumbers) // 24
```
