<LearnSectionContent>
Basic arithmetic operators:

- \+
- \-
- \*
- /
- %
- ++
- \-\-

Example:

</LearnSectionContent>

```javascript
let age = 20
let addOneToAge = age + 1
let subtractOneFromAge = age - 1

console.log(age + 3) // 23
console.log(age / 4) // 5
console.log(age * 2) // 40
console.log(age % 3) // 2
```
